== КУРС: Frontend-разработчик ==

Название: Frontend-разработчик с нуля до Junior
Описание: Полный курс по созданию современных веб-сайтов и интерфейсов. Ты освоишь HTML, CSS, JavaScript и научишься работать с популярным фреймворком React. Курс построен на реальных проектах — к выпуску у тебя будет портфолио из 5 работ. Преподаватели — практикующие разработчики из Яндекса и других крупных компаний.
Стек технологий: HTML5, CSS3, JavaScript (ES6+), React 18, TypeScript, Git, Figma (чтение макетов), REST API, Webpack/Vite
Длительность: 9 месяцев
Цена: 89 900 ₽ (полная оплата) / 7 490 ₽ в месяц (рассрочка на 12 месяцев, без переплат)
Формат: Онлайн — живые вебинары 3 раза в неделю + записи всех занятий
Для кого: Идеально подходит для новичков без опыта в программировании; также берут тех, кто знает базовый HTML/CSS
Первый результат: Через 4 недели сверстаешь свою первую веб-страницу по макету
График: Гибкий — все записи доступны 24/7, но живые вебинары по вечерам (19:00–21:00 МСК)
Трудоустройство: Карьерный центр помогает с резюме, портфолио и поиском работы

== КУРС: Python-разработчик ==

Название: Python-разработчик: от основ до Backend
Описание: Курс охватывает Python с нуля и доходит до создания полноценных backend-приложений на Django и FastAPI. Ты научишься работать с базами данных, строить REST API и деплоить проекты на облачных серверах. В конце курса создашь дипломный проект — работающий веб-сервис, который можно показать работодателю. Преподаватели — backend-разработчики из крупных технологических компаний с реальным опытом проектной работы.
Стек технологий: Python 3.11, Django 5, FastAPI, PostgreSQL, Redis, Docker, Git, SQLAlchemy, Pydantic
Длительность: 10 месяцев
Цена: 94 900 ₽ (полная оплата) / 7 910 ₽ в месяц (рассрочка на 12 месяцев)
Формат: Онлайн — живые вебинары 2 раза в неделю + записи + практические задания с проверкой куратором
Для кого: Подходит полным новичкам; есть ускоренный трек для тех, кто уже знает базовый Python
Первый результат: Через 3 недели напишешь первую программу, автоматизирующую реальную задачу
График: Фиксированный старт (наборы 1-го числа каждого месяца), но занятия можно смотреть в записи
Трудоустройство: Помощь с поиском работы, партнёрские вакансии от 30+ компаний

== КУРС: UI/UX дизайнер ==

Название: UI/UX дизайнер: создавай интерфейсы, которые продают
Описание: Курс научит тебя проектировать удобные и красивые цифровые продукты — от мобильных приложений до веб-сервисов. Ты освоишь Figma на профессиональном уровне, научишься проводить UX-исследования и защищать свои решения перед командой. Курс делает акцент на практике: 70% времени — работа над реальными кейсами. Преподаватели — практикующие дизайнеры из известных digital-агентств и продуктовых IT-компаний.
Стек технологий: Figma, FigJam, Principle (прототипирование), Maze (юзабилити-тестирование), основы HTML/CSS для взаимодействия с разработчиками
Длительность: 7 месяцев
Цена: 79 900 ₽ (полная оплата) / 6 660 ₽ в месяц (рассрочка на 12 месяцев)
Формат: Онлайн — живые воркшопы 2 раза в неделю + разборы работ в группе
Для кого: Подходит как полным новичкам в дизайне, так и графическим дизайнерам, которые хотят перейти в digital
Первый результат: Через 2 недели создашь свой первый прототип мобильного экрана в Figma
График: Гибкий — воркшопы записываются, задания можно сдавать в удобное время
Трудоустройство: Помощь в формировании портфолио, разборы с практикующими арт-директорами

== КУРС: QA-инженер ==

Название: QA-инженер: тестировщик программного обеспечения
Описание: Ты освоишь профессию специалиста по качеству ПО — одну из самых востребованных и доступных для входа в IT. Курс охватывает ручное и автоматизированное тестирование, работу с документацией, баг-трекерами и API. Ты пройдёшь путь от написания первых тест-кейсов до автотестов на Python — чтобы расти в профессии без смены направления. Преподаватели — действующие QA-инженеры и тест-менеджеры из крупных IT-компаний.
Стек технологий: Postman, Jira, TestRail, Charles Proxy, SQL (базовый уровень), Python (для автотестов), Selenium/Playwright, Git
Длительность: 6 месяцев
Цена: 69 900 ₽ (полная оплата) / 5 830 ₽ в месяц (рассрочка на 12 месяцев)
Формат: Онлайн — живые вебинары 2 раза в неделю + практика на реальных тестовых стендах
Для кого: Идеально для новичков; особенно подходит тем, кто внимателен к деталям и любит порядок
Первый результат: Через 2 недели научишься писать тест-кейсы и находить баги в готовых приложениях
График: Гибкий, с несколькими потоками старта в месяц
Трудоустройство: Карьерная поддержка, помощь с первым резюме, база стажировок
